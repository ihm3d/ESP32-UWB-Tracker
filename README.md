# ESP32 UWB Tracker (REYAX RYUW122 Lite)

This project demonstrates how to build a simple UWB distance tracker using two REYAX RYUW122 Lite modules and an ESP32.

## Features

- OLED display
- AT command communication
- Distance measurement
- Distance filtering
- ESP32 source code
- Based on REYAX RYUW122 Lite

## Hardware

- ESP32
- REYAX RYUW122 Lite ×2
- SSD1306 OLED Display
- CP2102 / FTDI
